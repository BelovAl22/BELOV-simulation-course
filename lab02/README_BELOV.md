### Метод конечных разностей для уравнения теплопроводности

**Задание:**  
Реализовать моделирование изменения температуры в пластине на основе одномерного уравнения теплопроводности с использованием метода конечных разностей.

Выполнить моделирование с различными шагами по времени и по пространству.  
Заполнить таблицу значений температуры в центральной точке пластины после 2 секунд модельного времени.

Материал: сталь, толщина пластины: 0,03м

Метод прогонки показывает устойчивую сходимость при умеренных шагах: при h = 0,001 м температура в центре стабилизируется на уровне ~77,8*, но колеблется в зависимости от шага по времени.
Крупный шаг (h = 0,01м) дает завышенное значение ~82,1* из-за недостаточно детализации сетки. При очень мелких шага (h <= 0,001) наблюдается численная неустойчивость, значения колеблются.
Таким образом, оптимальным вариантом является h = 0,001 и t = 0,01, т.к. этот вариант дает аланс между точностью, устойчивостью и скоростью расчета.
